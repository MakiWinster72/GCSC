# 后台管理

「后台管理」是系统的超级管理员面板，提供用户管理与数据备份功能。

> 仅 **管理员** 可访问此页面。

## 用户管理

![用户管理页面](/images/admin用户管理.png)

### 查看用户列表

以表格形式展示所有注册用户，列包括：

- **用户名**
- **显示名称**
- **角色**（学生 / 班干部 / 教师 / 管理员）
- **注册时间**
- **最后登录时间**
- **账号状态**（正常 / 禁用）

### 编辑用户角色

点击用户行的「编辑」按钮，可修改用户角色。

角色说明：

| 角色 | 说明 |
|---|---|
| STUDENT（学生） | 普通学生，可使用全部个人功能 |
| CADRE（班干部） | 可审核本班成员的信息修改 |
| TEACHER（教师） | 可查看学生档案 |
| ADMIN（管理员） | 拥有全部权限 |

### 禁用/启用账号

点击「禁用」可将账号临时关闭，被禁用用户无法登录。

再次点击「启用」可恢复账号。

### 重置密码

点击「重置密码」，系统生成随机密码并展示，管理员可告知用户。

## 数据备份

### 导出数据

支持导出以下数据为 Excel 文件：

- 全部学生档案信息
- 全部成果记录

点击「导出」按钮，选择导出范围，确认后下载文件。

### 备份说明

- 导出数据包含所有字段
- 成果附件（图片、视频等）不在导出范围内
- 建议定期进行数据备份，防止数据丢失

## 注册开关

管理员可开启或关闭注册功能。

![其他设置页面](/images/admin其他设置.png)

关闭后，新用户将无法访问注册页面，已有账号不受影响。
