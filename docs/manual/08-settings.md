# 账号设置

「账号设置」用于管理个人账号安全相关选项。

![设置页面](/images/settings设置页.png)

所有角色均可访问此页面。

## 修改密码

### 操作步骤

1. 输入**当前密码**（验证身份）
2. 输入**新密码**（建议 8 位以上，含字母和数字）
3. 再次输入**确认新密码**
4. 点击「确认修改」

### 注意事项

- 新密码不能与当前密码相同
- 确认密码需与新密码完全一致
- 修改成功后需重新登录

## 登出

点击「退出登录」按钮，确认后清除本地登录状态，跳转到登录页面。

> 在公共设备上使用系统后，务必点击退出登录。

## 账号信息

页面顶部显示当前登录账号的基本信息（不可编辑）：

- **用户名**
- **显示名称**
- **角色**
- **注册时间**
