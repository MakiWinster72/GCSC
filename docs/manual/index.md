# 使用手册

系统内置四种角色，对应不同的访问权限：

| 角色       | 说明         | 可访问页面                     |
| ---------- | ------------ | ------------------------------ |
| **学生**   | 普通学生账号 | 我的信息、个人成果、通知、设置 |
| **班干部** | 班级审核权限 | 学生权限 + 班级审核            |
| **教师**   | 查看学生档案 | 班干部权限 + 学生信息          |
| **管理员** | 系统后台管理 | 全部页面                       |

## 页面索引

### 通用页面

- [登录与注册](./01-login) — 账号登录、注册、找回密码
- [我的信息](./02-myinfo) — 个人档案信息查看与编辑
- [个人成果](./03-achievements) — 成果录入、编辑与提交
- [通知中心](./04-notifications) — 审核状态、通知消息
- [账号设置](./08-settings) — 修改密码、登出

### 管理页面

- [学生信息管理](./05-student-info) — 按班级查看学生档案（教师/管理员）
- [班级审核](./06-class-reviews) — 审核班级成员提交的信息（班干部）
- [后台管理](./07-admin) — 用户管理、数据备份（管理员）
