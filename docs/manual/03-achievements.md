# 个人成果

「个人成果」用于录入、查看和管理个人各类成果。

## 成果类型

系统支持以下九类成果，每类独立管理：

| 类型 | 说明 |
|---|---|
| 竞赛 | 学科竞赛、创新创业比赛等 |
| 科研 | 参与教师科研项目 |
| 论文 | 发表的期刊论文、会议论文 |
| 专利 | 发明专利、实用新型专利、外观设计专利 |
| 证书 | 专业技能证书、语言证书等 |
| 作品 | 艺术作品、软件作品、设计作品等 |
| 期刊 | 期刊投稿与发表 |
| 双百 | 双百计划相关成果 |
| 培训 | 参与的培训项目 |

## 查看成果列表

顶部标签栏切换不同成果类型，默认显示「全部」。

![个人成果总览页](/images/achievements个人成果总览页.png)

每个成果卡片显示：

- **成果名称**
- **提交时间**
- **审核状态**（待审核 / 已通过 / 已驳回）
- **媒体附件数量**

## 新增成果

点击「新增」按钮，选择成果类型，填写表单：

| 字段 | 说明 |
|---|---|
| **成果名称** | 必填，简明扼要 |
| **详细描述** | 成果内容、角色、收获等 |
| **媒体附件** | 图片、视频、PDF 文档（最多 9 个） |

附件支持格式：图片（jpg/png）、视频（mp4）、PDF、Word、Excel，单个文件不超过 200MB。

![添加成果页面](/images/achievements个人成果-添加成果.png)

## 编辑与删除

- 点击任意成果卡片进入详情页

![查看成果详情](/images/achievements个人成果查看.png)

- 在详情页可「编辑」修改内容，或「删除」整条记录
- 删除前系统会二次确认

## 提交审核

成果编辑完成后，点击「提交审核」将成果送交审核流程。

> 审核通过前，成果状态为「草稿」；提交后变为「待审核」。

## 审核状态说明

- **草稿**：未提交，仅自己可见
- **待审核**：已提交，等待审核
- **已通过**：审核通过，纳入统计
- **已驳回**：审核未通过，可查看驳回理由并重新编辑提交
