# 通知中心

「通知中心」集中展示所有审核状态更新和系统消息。

![通知中心](/images/notification审核.png)

## 通知类型

| 类型 | 说明 |
|---|---|
| 成果审核 | 成果提交后的审核结果通知 |
| 信息修改 | 个人资料变更的审核结果通知 |
| 系统通知 | 管理员发布的公告 |

## 通知列表

按时间倒序展示，最新通知在最上方。

每条通知显示：

- **类型图标**：成果类 / 信息类 / 系统类
- **标题**：如「成果已通过审核」「信息修改已驳回」
- **时间**：发送时间
- **状态**：未读 / 已读

## 通知详情

点击通知卡片展开详情：

- **成果审核通知**：展示成果名称、审核结果（通过/驳回）、审核意见
- **信息修改通知**：展示修改字段、原值 → 新值、审核意见
- **系统通知**：展示公告正文

## 审核状态与成果页同步

在通知中心点击成果相关通知，可直接跳转到「个人成果」页面查看对应成果详情。

## 未读标记

收到新通知后，顶部导航栏的通知入口会显示红色未读计数。

进入通知中心后，未读通知自动标记为已读。
