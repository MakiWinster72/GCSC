# 操作日志

> 仅 **管理员** 可访问此页面。记录系统内高敏感操作的审计追踪。

## 功能概述

操作日志记录以下类型的操作：

| 操作类型 | 说明 |
| -------- | ---- |
| 导出数据库 | 管理员通过后台管理导出 SQL 备份文件 |
| 恢复数据库 | 管理员通过后台管理恢复 SQL 备份 |
| 导出附件 | 管理员通过后台管理导出 ZIP 附件包 |
| 恢复附件 | 管理员通过后台管理恢复 ZIP 附件 |
| 导出学生信息 | 教师/管理员在学生信息页面导出学生数据 |
| 通过成就审核 | 审核人员通过学生的成就修改申请 |
| 驳回成就审核 | 审核人员驳回学生的成就修改申请 |
| 通过信息审核 | 审核人员通过学生的个人信息修改申请 |
| 驳回信息审核 | 审核人员驳回学生的个人信息修改申请 |

## 页面结构

管理员在侧边栏导航中点击「操作日志」进入。

![操作日志页面](/images/logs-page.png)

### 搜索

输入用户名、操作类型或详情关键字进行筛选，支持模糊匹配。

### 日志列表

每条日志包含：

| 列 | 说明 |
| -- | ---- |
| 时间 | 操作发生的时间（精确到秒） |
| 用户 | 执行操作的用户名 |
| 操作 | 操作类型（中文标签） |
| 详情 | 操作的具体描述，驳回类操作包含驳回理由 |
| IP | 操作者的客户端 IP 地址 |

### 分页

列表支持分页浏览，默认每页 20 条。

## 注意事项

- 操作日志为只读，不可删除或修改
- 日志由系统自动记录，无需手动创建
