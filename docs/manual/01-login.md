# 登录与注册

## 登录

访问系统后，未登录用户直接进入登录页面。

### 输入凭证

| 字段 | 说明 |
|---|---|
| **用户名** | 4–32 位英文字母、数字或下划线 |
| **密码** | 账号密码 |

![登录页面](/images/login登录页面.png)

### 登录后

登录成功后，系统根据账号角色自动跳转到对应首页：

- **学生 / 班干部** → 「我的信息」页面
- **教师** → 「学生信息管理」页面
- **管理员** → 「后台管理」页面

### 错误处理

- 用户名或密码错误：页面提示"用户名或密码错误"，清空密码框，请重新输入
- 账号被禁用：提示"账号已被禁用"，请联系管理员
- 网络异常：提示网络错误，可重试

---

## 注册

在登录页面点击「没有账号？去注册」进入注册页面。

### 输入信息

| 字段 | 说明 |
|---|---|
| **用户名** | 4–32 位英文字母、数字或下划线，唯一不可重复 |
| **显示名称** | 姓名，用于界面展示 |
| **密码** | 建议包含字母与数字 |
| **角色** | 选择：学生（默认）、教师、班干部 |

![注册页面](/images/register注册页面.png)

> 注册功能由管理员控制开启/关闭。若页面无法访问，请联系管理员。

### 注册成功

注册成功后自动登录并跳转到首页。

---

## 安全提示

- 请勿将账号密码告知他人
- 公共设备使用后请及时登出
- 发现异常登录请立即修改密码
