# 登录与注册

## 登录

![登录页面](/images/login登录页面.png)

访问系统后，未登录用户会进入登录页面。

> 管理员可以在 `后台管理` > `其他设置` 里面关闭/开启注册功能.

### 输入凭证

| 字段                     | 说明                          |
| ------------------------ | ----------------------------- |
| **用户名(一般就用学号)** | 4–32 位英文字母、数字或下划线 |
| **密码**                 | 账号密码                      |

登录后默认跳转到 `我的信息(myinfos)` 界面

## 注册

在登录页面点击「没有账号？去注册」进入注册页面。

![注册页面](/images/register注册页面.png)

### 输入信息

| 字段     | 说明                                        |
| -------- | ------------------------------------------- |
| **名字** | 姓名，用于界面展示                          |
| **学号** | 4–32 位英文字母、数字或下划线，唯一不可重复 |
| **密码** | 建议包含字母与数字                          |

### 注册成功

注册成功后跳转到登录页登录。

首次登录会提示先填写[我的信息](./02-myinfo.md).
